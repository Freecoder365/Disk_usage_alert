# Disk Usage Alert Script v2

## Description
This project is a Bash-based disk monitoring script that checks disk usage for a given mount point and triggers alerts when usage exceeds a defined threshold.

The script also logs system status and alert activity, making it useful for basic monitoring and automation tasks in Linux environments.

---

## Features
- Accepts mount point, threshold, and email as input arguments
- Checks disk usage dynamically
- Sends alert when usage exceeds threshold
- Logs system activity to a file
- Reusable and configurable script

---

## Technologies Used
- Bash (Shell Scripting)
- Linux Commands: `df`, `awk`, `tr`, `date`
- Mail utility (for alerts)
